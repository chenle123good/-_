# -_2.模型介绍
VGG有两种结构，分别是VGG16和VGG19，两者并没有本质上的区别，只是网络深度不一样。本项目采用VGG19。VGG16相比AlexNet的一个改进是采用连续的几个3x3的卷积核代替AlexNet中的较大卷积核。具体使用了3个3x3卷积核来代替7x7卷积核，使用了2个3x3卷积核来代替5*5卷积核，与VGG16相比，VGG19就是每3层卷积多了一个3*3的卷积层。
